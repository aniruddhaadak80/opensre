# Tracer Community Rules

Welcome to the Tracer open-source SRE observability community! We strive to create a welcoming, inclusive, and highly collaborative environment. These rules (modeled after the GitLab Code of Conduct) apply to our Discord server, GitHub repositories, mailing lists, and community calls.

## Expected Behavior
- **Respect and Inclusivity:** Treat everyone with respect. We welcome contributors from all backgrounds.
- **Assume Good Intent:** Approach interactions with the assumption that others are trying to help.
- **Constructive Technical Critique:** When reviewing code or designs, critique the work, not the person. Be kind and clear.

## Unacceptable Behavior
The following behaviors will not be tolerated:
- Harassment, discrimination, or intimidation of any kind.
- Spam, unsolicited off-topic promotion, or selling products/services.
- Doxxing or publishing others' private information without explicit permission.

## Channel Etiquette
- **Use Threads:** Keep conversations organized. Reply in threads instead of the main channel.
- **Search Before Asking:** Please check the documentation, pinned messages, or previous threads before asking a question.
- **No DM-for-help:** Do not direct message maintainers for technical support. Keep discussions in public channels so everyone can learn.

## AI-Generated Content / Disclosure Norms
Given that Tracer is an agent-based project, the use of AI-generated content is frequent.
- Please clearly disclose when a large block of text or code you are sharing was entirely AI-generated.
- Do not use AI to bulk-generate low-quality pull requests or comments.

## Reporting & Enforcement
If you experience or witness a violation of these rules, please reach out to the core maintainers via community@tracer.cloud or direct message a server admin on Discord.
- **Response SLA:** We commit to reviewing reports within 48 hours.
- **Escalation:** Violations may lead to a warning, a temporary mute/ban, or a permanent ban from all Tracer community spaces, depending on severity.

Thank you for helping us build a great community!
